## 什么是Nezha Mobile

Nezha Mobile 是 [Nezha Dashboard](https://nezha.wiki) 的非官方客户端，将服务器监控带到 Apple 全生态。随时随地查看服务器状态、实时指标，管理告警规则，并接收推送通知。

## 支持的平台

Nezha Mobile 是一款为所有 Apple 平台原生开发的通用 App：

— iPhone 与 iPad（iOS / iPadOS 17.0+）
— Mac（macOS 14.0+）
— Apple Watch（watchOS 10.0+）
— Apple Vision Pro（visionOS 1.0+）

## 功能

### 实时掌握每一台服务器

- **实时服务器列表** — 每个节点的 CPU、内存、Swap、磁盘以及上下行网速，全部以秒级刷新。
- **深入的服务器详情** — 负载平均、运行时长、进程数、地理位置以及历史走势图，任何异常都能一探到底。
- **世界地图视图** — 全部服务器按地理位置在世界地图上聚合呈现，区域故障一眼可见。

### 抢在故障之前一步

- **告警规则** — 在 iPhone 上即可创建、编辑、调整 CPU、内存、磁盘、流量等阈值。
- **服务监控** — 跟踪 HTTP、TCP 与 ICMP 服务的可用性与历史，一点即看。
- **推送通知** — 通过 APNs，告警在触发的瞬间送达你的 iPhone、Mac 与 Apple Watch。

### 随时随地管理你的集群

- **通知与通知组** — 在 App 内配置通知通道并将告警精准路由，无需离开当前界面。
- **服务器分组** — 按环境、地区、客户或任意维度组织服务器 —— 你说了算。
- **代码片段** — 命令与配置片段一次保存，随时随地一键复用。

### 全平台原生体验

- **无处不在的小组件** — 集群汇总、单服务器详情、Agent 小组件可放置在主屏幕、锁屏、待机模式与 Mac 桌面，并通过 App Intents 一键刷新。
- **iCloud 同步** — 一次登录，仪表盘凭据与偏好自动出现在你的每一台设备，密钥安全保存在 Keychain。
- **主题** — 流畅的 SwiftUI 设计，自动适配你的浅色、深色与强调色偏好。
